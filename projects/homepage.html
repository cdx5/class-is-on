• Homepage •

Create a personal website to house the individual works you will create in this class. Your page should have a considered design that expresses your visual voice. You may use any combination of text, colors, and images. There are no limitations on the content of your site, the only requirement is to include a container which will eventually link to each of this course's projects to come.

Establish an organized folder structure for this class:

my-homepage/
├─ index.html
├─ css/
│  ├─ style.css
├─ projects/

In this model each of your future assignments for this class can be stored in the "projects" folder.

Publish your homepage using your GitHub account and [GitHub Pages]. You may use [GitHub Desktop] to upload and manage your files if you are not comfortable using command line. Email me the link to your live site by September 16. <abber@newschool.edu>
